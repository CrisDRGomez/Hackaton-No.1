Checklist – Landing Page Hackathon 1
Requisitos que debe contener el proyecto

1. Estructura de la página (HTML)
La landing page debe tener mínimo estas secciones:
Navbar / Menú
Hero o Banner principal
Sección de productos
Sección de información / beneficios / ofertas
Sección de contacto (formulario o información)
Footer

2. Contenido de la página
La página debe incluir:
Nombre de la tienda
Imágenes de productos
Nombre de productos
Precios
Botones (comprar, agregar, ver más)
Textos descriptivos
Logo o título
Formulario o información de contacto

3. CSS (Diseño)
Debe tener estilos, por ejemplo:
Colores
Tipografías
Cards de productos
Navbar estilizado
Botones estilizados
Hover en botones o productos
Diseño organizado
Uso de Flexbox o Grid
Página centrada y ordenada

4. JavaScript (Interacción)
Debe tener al menos una interacción con JavaScript, por ejemplo:
Botón agregar al carrito → alert
Cambiar color al hacer click
Mostrar mensaje
Animación
Menú responsive
Modal
Contador
Slider
Mostrar/ocultar información
Mínimo 1 o 2 interacciones JS.

5. GitHub
El proyecto debe:
Estar en un repositorio de GitHub
Tener commits
Tener README
Tener todos los archivos del proyecto
Trabajar en equipo (si aplica)
